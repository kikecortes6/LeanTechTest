# LeanTechTest


## Correr el proyecto backend

Para correr el **backend** primero navegamos a la carpeta del proyecto después para instalar las dependencias necesarias 

```bash
mvn clean install
```

y luego para correr el proeycto ejecutar desde la consola

```bash
 mvn spring-boot:run
```
